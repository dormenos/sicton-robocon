Танилцуулга

Бидний багт механик зохион бүтээгч, схем дизайнч, суурь программист хүртэл цогцолж — нэг зорилгод нэгддэг.

  • Механик: CAD (Fusion 360), 3D хэвлэл, CNC
  • Электроник: STM32, ESP32, мотор драйвер, мэдрүүр
  • Програм: C/C++, Python, ROS2, OpenCV/YOLO
  • Сорил: Unit test, HIL, талбайн туршилт

ABU Robocon юу вэ?

ABU Robocon нь Ази–Номхон далайн орны инженер оюутнууд робот бүтээж, тухайн жилийн дүрмээр тэмцдэг тэмцээн. Жил бүрийн даалгавар өөр сэдэвтэй байдаг.

  1. Тухайн жилийн дүрэм, онооны системийг судална
  2. Стратеги сонголт: хурд, нарийвчлал, найдвартай байдал
  3. Роботын архитектур: механик + электроник + софт
  4. Талбайн дасгал, параметр тюнинг, лог шинжилгээ

Манай зорилго: Fast, Robust, Repeatable.

Роботууд

Автомат & Гар удирдлагатай роботуудын богино танилцуулга.

Autonomous

Orion A1

STM32F407 + Jetson Orin Nano, ROS2 Humble, дифференциал жолоодлого, YOLOv8 дүр танилт, киллер фичер — 100ms доорх pipeline.

  • Drive: Omni/Дифференциал (модульчлагдсан)
  • Perception: RGB-D + IMU + одометр
  • Control: PID + LQR тюнинг
Manual

Atlas M2

ESP32 + PS5 controller (BLE), VNH5019 драйвер, хурд/эргэлтийн хаалттай цикл; хурдан, хүчтэй, эвдэрдгийг засахад хялбар.

  • Torque: 24V BLDC + planetary gearbox
  • Safety: E-stop, current limit
  • Swap: 3 минутын модуль солих
